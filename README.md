# JRB_DS_2026_GRUPO_01
### Professor Orientador Fábio D. Sinobre
#### Alunos:
Projeto integrador do curso de Desenvolvimento de Sistemas da Escola Estadual Juvenal Ramos Barbosa
Stack: Python, Flask, Jinja2, Agile (Scrum/Kanban).
Descrição: Desenvolvimento do MVP (Produto Mínimo Viável) de uma plataforma web interativa. Este repositório documenta a jornada de aprendizagem técnica do primeiro ano, cobrindo desde a arquitetura de rotas e templates até a gestão de tarefas via Kanban. O projeto simula o fluxo de trabalho de um escritório de software real, priorizando código limpo, documentação eficiente e colaboração via GitHub Codespaces.
